# Problem Solving Journal
---
#This is not a streak-based repository.!!!
---

This repository tracks my structured practice of Data Structures & Algorithms in Java.

Instead of just solving problems, the goal here is to:

- Re-solve problems after a gap to test retention
- Write my own explanations and reasoning
- Analyze time & space complexity
- Track mistakes and edge cases
- Organize problems topic-wise for revision

- `README.md` — Explanation, approach, complexity analysis, and reflections
